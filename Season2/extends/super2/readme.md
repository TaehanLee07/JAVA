#   클래스와 메서드에 사용되는 final
#    1. 클래스에 final
    - 상속 끝
    - ㅇfinal로 선언되면 확장할 수 없다. 다른 클래스가 final로 선언된 클래스를 상속받을수 없다.
    - public final class MyFinalClass() {...}
#    2. 메서드에 final
    - 오버라이딩 끝
    - final로 선언된 메서드는 오버라이드 할 수 없다. 상속받은 서브 클래스에서 이 메서드를 변경 할 수 없다.
    - public final void MyFinalMethod() {...}
